Fabrication

All the STL files of Humanoid are made available in the repo's Assests folder. Fabricate the parts using 3D printer or CNC machine (Recomended). In my robot i have used aluminium material. Assemble the chassis by refering the images in showcase page.

Final Assembly

Servo Motor

I have used high torque servo motor for actuation. The dimensions of the servo are 40 x 20 x 40.5mm (height, length, width). Kindly check the motor's dimension before buying. Make sure the motors have full metal gears. Even partial plastic leads to breakage of motor. Check whether the motor rotates or not before connecting to the chassis.

Servo Motor

Motor Controller

Any servo controller which can drive more than 17 servos will be suitable for our application. Make sure all motors receive enough voltage when connected. Here a PCB is integrated with servo driver, bluetooth and two arduino microcontroller is used.

Servo Controller

Processor

For running ROS (Robot Operating System) and deep learning models a processor (Computer) is required. I have used my laptop for it. Instead of that Raspberry Pi or Jetson nano can also be used if you have enough budgets.

Computer (processor)

Schematics

The schematics is the very simplest part in our humanoid.

  1. Use a USB wire to connect the computer and arduino or any development board.
  2. Connect all the servo wires to the servo driver.
  3. Use a shield or something to join the development board and the motor driver together. (In our case it is not required).
  4. Connect the battery to the power port and join all the ground together.

Wiring

Power Supply

Use a original Orange LiPo 12V battery and a balanced charger for recharging.

Deploy

Create a ROS Package

$ cd "your-catkin-workspace"/src 
$ catkin_create_pkg Humanoid-Robot rospy

Make Scripts Executable

$ roscd Humanoid-Robot/src 
$ chmod +x *

Burn Code to Arduino

Arduino IDE

  1. Compile humanoid_debug.ino code.
  2. Then select Tools ‚û°Ô∏è Board ‚û°Ô∏è Arduino UNO from menu bar.
  3. Repeat the above steps to select port.
  4. Finally upload the code to the development board.

Run Script

Run each command in a seperate terminal.

$ roscore
$ rosrun Humanoid-Robot debug.py
$ rosrun rosserial_python serial_node.py
$ rqt_graph #Run this command to visualize all active nodes

Enter servo number (range 1 to 18) and then its position (degree) (range 0 to 180) in 2nd terminal (debug.py)
